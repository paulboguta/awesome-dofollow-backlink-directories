# Contributing

Thanks for considering a contribution.

## What gets accepted

- **Active directories** — site loads, accepts submissions in 2026.
- **Quality directories** — real audience, real backlinks (dofollow preferred), not a PBN.
- **In-scope** — fits the theme of this list. Off-topic submissions will be redirected to the right list.

## How to submit

1. Open a PR adding your directory to the right section.
2. Use the existing format: `- **[Name](url)** · DR XX · dofollow · free — Short note.`
3. Sort entries within a section by DR (highest first).
4. If you don't know DR/DA, leave it blank — the maintainer will check.

## What gets rejected

- Link farms, PBNs, recently spun-up domains with no traffic
- Directories that charge $500+ with no audience
- Duplicates (use search before adding)
- Affiliate-only or paywalled submission forms with no real audience
